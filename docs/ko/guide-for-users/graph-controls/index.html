그래프 제어 기능

이전 페이지에서는 단일 곡선에 적용되는 제어 기능을 다루었습니다. 이 페이지는 그래프 전체에 적용되는 제어 기능을 다룹니다. 이 제어 기능들은 두 곳에 위치합니다:

  • 그래프 패널: 최상단에 타겟 선택기가 있고 그 아래에 선택 목록이 있습니다.
  • 그래프 툴바: 데스크톱에서는 _그래프 바로 아래_에 위치하며, 모바일에서는 그래프 패널 상단의 접이식 UI 안에 들어 있습니다.

타겟 선택기​

그래프 패널 최상단에는 타겟 선택기가 있습니다. 사이트가 제공하는 모든 타겟이 이 목록에 모여 있습니다. 하나를 선택하면 그래프에 타겟 커브가 추가되며, 보통 회색 점선으로 표시됩니다.

"뉴트럴"을 정의하는 두 타겟을 나란히 비교하고 싶다면 한 번에 두 개 이상도 선택할 수 있습니다. 타겟은 사이트 설정에서 그룹으로 묶이므로, 실제로 표시되는 목록은 배포 환경마다 다릅니다.

타겟 철학(왜 이렇게 많은 타겟이 존재하고 각각이 무엇을 의미하는지)은 타겟이 존재하는 이유를 참조하십시오. 타겟의 형태를 조정하는 방법은 타겟과 선호도를 참조하십시오.

정규화​

정규화는 툴이 각 곡선을 수직으로 어디에 놓을지 결정하는 방식입니다. 정규화가 없다면 서로 다른 전체 볼륨으로 측정된 두 기기가 그래프에서 전혀 다른 높이에 놓여 비교가 어렵습니다. 정규화는 곡선들을 직선 방향으로 정렬해 형태의 차이를 또렷이 드러냅니다.

툴바에는 두 가지 모드를 가진 분할 토글이 있습니다:

Avg 모드 (기본)​

300 Hz와 3,000 Hz 사이의 음압(SPL)을 평균 내고, 그 평균을 각 곡선의 영점으로 삼습니다. 불러온 모든 곡선이 동일한 중음역 평균을 가지게 되며, 가청 대역의 가장 핵심적인 부분을 가로질러 곡선들을 자연스럽게 정렬합니다.

일반적인 비교에는 Avg를 권장합니다. 대부분의 상황에서 가장 적절한 기본값입니다.

Hz 모드​

평균 대신 단일 주파수를 선택하면, 모든 곡선이 해당 주파수에서 0 dB이 되도록 이동합니다. 토글 옆 주파수 입력으로 정확한 값(20에서 20,000 사이의 모든 정수 Hz)을 선택할 수 있습니다.

특정 대역에서 두 곡선을 비교할 때는 Hz를 사용하십시오. 예를 들어 "이 두 기기를 500 Hz에서 맞추면 저음에서 얼마나 차이가 날까?"의 답은 다른 모든 주파수에서의 수직 거리입니다.

정규화는 데이터 변경이 아닌 화면 설정입니다

어느 모드도 원본 측정 파일을 수정하지 않습니다. Avg와 Hz 사이의 전환은 같은 데이터를 다른 정렬 기준으로 다시 그리는 것뿐입니다.

스무딩​

측정 파일에는 미세한 디테일이 많이 담겨 있고, 그중 상당수는 실제 청감에 영향을 주지 않는 노이즈나 측정 환경에 따른 굴곡입니다. 스무딩은 각 데이터를 인접한 데이터들과 평균 내어 더 깔끔한 곡선을 만듭니다.

스무딩 버튼은 1/48 옥타브(거의 스무딩이 없는, 사실상 원본 상태)부터 1/3 옥타브(강하게 스무딩된 상태)까지의 분수 옥타브 값을 순환합니다. 분모가 클수록 원본에 가깝고, 분모가 작을수록 더 부드러워집니다.

  • 1/48은 측정을 있는 그대로 보여줍니다. 노이즈를 포함한 모든 잔물결이 보입니다.
  • 1/24는 가장 흔히 쓰는 "미세한 디테일을 살리면서 형태를 보여주는" 설정입니다.
  • 1/6 또는 1/3은 곡선을 굵은 윤곽선으로 바꿉니다. 굴곡들이 화면에서 부딪치지 않아 여러 기기를 한 번에 비교할 때 유용합니다.

"정답"인 설정은 없습니다. 분석하려는 대상을 가장 보기 쉽게 해주는 설정을 고르십시오. 스무딩은 순전히 화면 설정이며, 기본 원본 파일은 그대로입니다.

Y축 스케일​

Y축 스케일 버튼(수직 양방향 화살표)은 Y축이 포함하는 dB 범위를 제어합니다. 범위가 작을수록 곡선이 확대되고, 범위가 클수록 축소되어 여러 오프셋 곡선을 쌓을 여유 공간이 생깁니다.

여기에도 단 하나의 정답은 없지만, 시작점으로 다음을 참고하십시오:

  • ~50 dB 범위: 기본 설정이자, 대부분의 비교에 적합한 범위입니다.
  • ~70 dB 범위: 공격적인 극저음 감소나 매우 높은 고음 피크를 분석할 때.
  • ~30 dB 범위: 비슷한 기기 사이의 미묘한 차이를 확대해서 볼 때.

과도한 확대는 주의하십시오. 20 dB 범위에서 거대해 보이는 5 dB 차이도 결국 5 dB이며, 시각적 차이가 곧 큰 청감상 차이를 뜻하지는 않습니다. 의심스러우면 축소하십시오.

조사 모드​

조사 모드(돋보기 아이콘)를 켜면 마우스가 그래프를 가로질러 움직이는 동안 작은 판독기가 마우스를 따라다니며 해당 위치의 정확한 주파수와 dB 값을 보여줍니다. 불러온 모든 곡선이 판독기 안에서 각자의 줄을 갖기 때문에, 한 번에 단일 주파수에서 모든 곡선을 비교할 수 있습니다.

모바일에서는 마우스를 올리는 대신 탭하고 드래그하십시오. 필요하지 않을 때는 조사 모드를 끄십시오. 그렇지 않으면 커서 추적이 화면을 가립니다.

선호도 경계 토글​

선호도 경계 버튼은 단일 타겟 선 대신 "허용 가능한" 튜닝의 통계적 범위를 나타내는 음영 영역을 켜고 끕니다. 자세한 내용은 선호도 경계를 참조하십시오. 이 버튼은 사이트가 경계 데이터 파일을 설정한 경우에만 활성화됩니다. 데이터 파일이 없으면 해당 배포 환경에서는 이 기능을 쓸 수 없습니다.

주파수 튜토리얼​

주파수 튜토리얼은 그래프의 각 영역—극저음, 저음, 중저음, 중음, 중고음, 초고음—의 이름을 명시하고 해당 영역에 무엇이 있는지 짧은 설명을 덧붙이는 라벨 오버레이입니다. 그래프 툴바 위쪽 영역에서 켜고 끌 수 있습니다. 노래의 어느 부분이 그래프의 어느 부분과 맞물리는지에 대한 감을 아직 익히는 중이라면, 이 오버레이를 한동안 켜두십시오. 전체 문서: 주파수 튜토리얼.

스크린샷, 공유, 그리고 쇼핑 링크​

툴바의 마지막 세 버튼은 내보내기 기능입니다. 자세한 사항은 해당 페이지에서 다룹니다:

  • 스크린샷 — 현재 그래프 화면을 PNG 이미지로 저장합니다.
  • 공유 — 다른 사람이 열어볼 수 있도록 전체 그래프 상태를 인코딩한 URL을 복사합니다.
  • 쇼핑 링크 — squig.link 사이트에서만 제공되며, squig.link를 지원하는 스폰서 데이터베이스에 해당 기기가 있으면 선택한 기기의 구매 페이지로 이동합니다.

공유와 내보내기를 참조하십시오.
