외관 및 언어

이 문서의 모든 내용은 데이터가 아닌 도구의 _외관_에 관한 것입니다. 측정값이나 EQ에는 어떠한 영향도 미치지 않으며, 화면에 데이터가 표시되는 방식만 변경합니다.

라이트 및 다크 테마​

기타 패널(네 번째 메뉴 아이콘, 데스크톱 환경에서는 숫자 키 4)을 엽니다. 상단의 해/달 아이콘을 클릭하면 라이트 모드와 다크 모드가 전환됩니다. 테마 전환 시 그래프를 포함한 전체 UI가 부드럽게 크로스페이드 처리됩니다.

선택한 테마는 브라우저 저장소에 기록되므로 다음 방문 시에도 동일하게 유지됩니다. 테마 설정을 변경한 적이 없다면, 사이트 설정에 따라 운영자가 지정한 기본 테마나 사용 중인 운영 체제의 라이트/다크 모드 설정이 초기 테마로 적용됩니다.

테마는 전적으로 시각적인 요소입니다. 다크 모드는 그래프 데이터, 곡선 색상, 위치 또는 내보낸 스크린샷 이미지에 아무런 영향을 주지 않습니다.

언어​

사이트에서 다국어 지원을 활성화한 경우 테마 전환 버튼 옆에 언어 선택기가 표시됩니다. modernGraphTool이 현재 지원하는 언어는 다음과 같습니다.

언어를 선택하면 UI 텍스트가 즉시 변경됩니다. 기기 이름, 타깃 이름, 운영자가 작성한 설명 텍스트는 번역되지 않으며 사이트 운영자가 데이터베이스에 입력한 원본 그대로 표시됩니다. 버튼 라벨, 패널 이름, 도움말 텍스트 등 공식 UI 문자열만 변경됩니다.

사이트에서 다국어 지원을 활성화하지 않았다면 언어 선택기는 숨겨지며, UI는 운영자가 설정한 기본 언어로 표시됩니다.

개별 곡선 색상​

각 선택 목록 행의 맨 앞에 있는 색상 견본을 클릭하면 소형 색상 선택기 팝업이 열립니다. 자동으로 할당된 색상 팔레트의 대비가 부족할 때 유용합니다. 예를 들어 로드한 두 헤드폰이 모두 비슷한 파란색으로 지정되어 곡선을 구별하기 어려울 때 활용할 수 있습니다.

해당 팝업에는 다음 항목이 포함되어 있습니다.

  • 클릭하여 운영 체제 내장 색상 선택기를 열거나 16진수 색상 코드를 직접 입력할 수 있는 네이티브 색상 입력란.
  • 정밀한 숫자 제어를 위한 HSL 입력란 (색조, 채도, 명도).
  • 곡선을 실선 대신 점선으로 렌더링하는 대시 패턴 입력란. 동일한 기기를 중복 로드하여 비교하거나 특정 관례에 맞춰 그래프 스타일을 통일할 때 유용합니다.
  • 새로운 색상을 무작위로 자동 지정하는 무작위화 버튼.

모든 변경 사항은 설정 즉시 그래프에 실시간으로 반영됩니다. 설정이 끝나면 팝업을 닫습니다. 지정한 색상과 대시 패턴은 공유 URL에 기록되므로, 특정 스타일을 적용한 상태로 링크를 공유하면 수신자도 동일한 화면을 볼 수 있습니다.

곡선을 초기 자동 할당 색상으로 되돌리려면 기기 패널에서 기기를 제거한 후 다시 로드하거나, 색상 선택기에서 새로운 색상을 직접 지정하면 됩니다.

주파수 튜토리얼​

주파수 튜토리얼은 그래프의 각 주파수 대역(서브 베이스, 베이스, 로우 미드, 어퍼 미드, 프레즌스, 브릴리언스)에 음악적 특성을 설명하는 짧은 라벨을 덧씌우는 오버레이 기능입니다.

그래프 툴바 상단 영역에서 해당 기능을 켜거나 끌 수 있습니다. 다음과 같은 상황에서 특히 유용한 보조 도구로 활용됩니다.

  • 주파수 응답 그래프 해석에 익숙하지 않은 경우.
  • 오디오 관련 용어에 능숙하지 않은 사람에게 헤드폰의 음색을 설명해야 하는 경우.
  • 특정 악기의 기음 대역을 X축 위에서 시각적으로 파악하고자 할 때 ("킥 드럼의 기음역대는 대략 _이곳_에 위치함").

라벨과 설명 텍스트는 사이트의 다국어 번역 파일에서 불러오므로, UI 언어 변경 시 함께 번역됩니다. 이 기능의 상세 작동 방식과 사이트 운영자의 텍스트 커스터마이징 방법은 주파수 튜토리얼 문서를 참조하십시오.

운영자 제공 콘텐츠​

기타 패널에는 사이트 운영자가 임의로 구성한 설명 콘텐츠도 표시됩니다. 보통 짧은 소개 글, 기여자 목록, 후원 링크 또는 관련 사이트 링크가 포함됩니다. 이는 형식이 자유로운 텍스트 영역이므로 배포된 사이트마다 보여지는 내용이 다릅니다.

기타 패널 하단에는 앱 버전, GitHub 저장소 및 공식 문서 링크가 표시되며, 운영자가 비활성화하지 않은 경우 프로젝트 개발자를 위한 후원 버튼이 함께 나타납니다. 해당 항목들은 개별 사이트 설정과 무관하게 모든 modernGraphTool 설치 환경에서 동일하게 제공됩니다.
