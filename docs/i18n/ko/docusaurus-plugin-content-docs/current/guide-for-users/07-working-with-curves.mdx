# 커브 다루기

불러온 모든 커브는 그래프 패널 내부의 **선택 목록**에 각각 행으로 표시됩니다. 특정 커브를 조절하려면 언제든지 그래프 패널(두 번째 메뉴 아이콘, 또는 데스크톱에서 **2** 키 입력)로 전환하십시오. 각 행의 레이아웃은 데스크톱과 모바일에서 동일합니다.

이 문서는 선택 기기 하나에 할당된 모든 조작 요소를 좌측 상단부터 우측 하단 방향으로 설명합니다. 전체 선택 목록 중에서도 타겟은 최상단에 우선 정렬되어 찾기 쉽도록 하였습니다.

## 선택 기기 레이아웃

하나의 선택 기기 항목은 2줄의 레이아웃으로 구성되어 있습니다.

- **상단 레이아웃:** 색상 버튼, 기기 이름(상세 항목이 있는 경우 화살표 표식 포함), 우측에 모여 있는 작업 버튼.
- **하단 레이아웃:** 채널 선택 버튼 및 세로축 위치 제어 버튼.

## 색상 버튼

좌측의 작은 사각형은 해당 기기의 그래프에 반영된 **색상**을 나타냅니다. 해당 버튼을 클릭하여 [색상 선택기](./12-appearance-and-language.mdx#per-curve-color)를 열 수 있습니다. 클릭 시 색상 선택 칸, HSL 입력란, 점선 패턴 제어 및 무작위 색상 버튼이 포함된 UI가 나타납니다. 여러 커브의 색상이 비슷하여 특정 커브 하나에 다른 색을 지정하고 싶을 때 사용할 수 있습니다.

그래프가 새롭게 추가될 때마다, 자동으로 다른 색상을 지정하려고 하므로 실제로 사용할 일이 많지는 않습니다.

## 이름 및 상세 항목 선택란

단일 측정 파일만 존재하는 기기에는 단순히 이름만 표시되는 반면, 이어팁, 이어패드, 착용 위치 등 **다양한 상세 항목**이 존재하는 기기는 이름 옆에 작은 화살표 기호가 표시됩니다. 이름을 클릭하면 모든 상세 항목이 나열된 목록이 열리며, 원하는 항목을 선택하면 현재 그래프의 커브가 해당 항목으로 **교체**됩니다.

두 항목을 동시에 그래프 위에 띄워두고 싶다면, 원하는 항목 오른쪽에 있는 **플러스 버튼**을 클릭하십시오.

상세 항목은 사이트 측정 데이터베이스에서 기기별로 구성됩니다. 기기에 상세 항목 데이터가 없다면 화살표 표식은 나타나지 않습니다.

## 기준선

눈 아이콘 옆에 있는 파형 형태의 **기준선** 버튼은 해당 커브를 기준점으로 고정하고 _다른 모든_ 커브를 이 기준점과의 차이 값으로 표시합니다. "FiiO 이어폰과 하만 타겟은 3 kHz 대역에서 어떻게 다른가?" 와 같은 의문을 해결할 때 유용합니다. 두 선을 수직으로 눈대중 비교하는 대신, 타겟을 기준선으로 설정하고 그래프에서 차이를 직접 읽어낼 수 있습니다.

기준선 설정 대상에 따라 두 가지 작동 방식이 있습니다:

**일반적인 헤드폰, 또는 사용자 맞춤화가 적용되지 않은 타겟:** 단순한 켜기/끄기 토글로 작동합니다. 한 번 클릭하면 켜지고, 다시 클릭하면 꺼집니다.

**선호도 조정 타겟** ([타겟 커스터마이저](./09-targets-and-preferences.mdx) 사용): 버튼은 세 가지 상태를 순환합니다. 따라서 기기 비교 분석과 분리하여, 사용자가 직접 수정한 튜닝 타겟의 효과를 독립적으로 확인할 수 있습니다.

| 상태 | 의미 |
|---|---|
| **꺼짐** | 일반 그래프 모드로, 기준선이 없습니다. |
| **조정 전** | 선호도 조정 이전의 _원본_ 타겟을 기준선으로 설정합니다. |
| **조정 후** | 선호도 조정값을 _포함한_ 타겟을 기준선으로 설정합니다. |

버튼 위에 커서를 잠깐 올려두면, 해당 커브가 현재 어느 상태에 있는지 텍스트로 알려줍니다. 클릭하면 다음 상태로 넘어갑니다.

:::info[기준선은 데이터 변환이 아닌 시각 보조 도구입니다]
기준선을 켠다고 해서 기본 측정 파일이나 EQ 파이프라인이 변경되지 않습니다. 단지 기존 커브가 그려지는 위치 좌표만 달라집니다. 기능을 끄면 원래 형태의 화면이 다시 나타납니다.
:::

## 표시 화면

**눈 아이콘**은 목록에서 커브를 삭제하지 않고 그래프 캔버스 위에서만 커브를 숨깁니다. 행이 어두워지며 숨겨진 상태임을 알립니다. 다시 클릭하면 화면에 나타납니다. 불러온 상태를 유지하면서 "_이 커브_를 제외하면 그래프가 어떻게 보일까"를 확인하고 싶을 때 유용합니다.

## 삭제

**휴지통 아이콘**은 커브를 완전히 제거합니다. **Ctrl+Z** 단축키를 눌러 이전 작업을 취소할 수 있으므로 실수로 클릭하더라도 복구할 수 있습니다.

## 채널 선택기

하단 레이아웃에는 채널 선택기가 위치합니다(타겟 그래프에는 표시되지 않습니다). 제공되는 측정 파일에 따라 옵션이 달라집니다:

- **L** — 왼쪽 채널 곡선만 그림
- **R** — 오른쪽 채널 곡선만 그림
- **L+R** — 두 곡선을 각각 분리된 선으로 그림
- **AVG** — 양쪽 채널을 하나의 선으로 평균 내어 그림

`L+R` 옵션이 제품 특성을 가장 상세하게 나타내며, `AVG` 옵션은 가장 간결하게 표현하는 방식이라고 할 수 있습니다. 측정 데이터가 단일 채널로만 제공될 경우 한 가지 옵션만 나타납니다.

### 샘플 선택기 (다중 샘플 기기용)

기기에 동일 모델을 여러 번 측정한 **다중 샘플** 데이터가 존재하는 경우, 더욱 다양한 설정이 포함된 **샘플/채널 선택기**가 기존 선택기를 대체하며 다음과 같은 기능이 추가됩니다.

- 특정 샘플 지정, 혹은 전체 샘플 평균 표시.
- 머리 전달 함수 편차를 **음영 대역**, **개별 커브**로 표시할지 혹은 두 가지 모두 표시할지 선택.

이를 통해 "이 헤드폰이 일관된 성능을 내는가, 개체 간 편차가 큰가"를 파악할 수 있습니다. 그래프 대역이 좁으면 편차가 적고, 넓으면 개체 간 특성 차이가 큼을 의미합니다.

## 세로 축 위치 제어

채널 선택기 우측에는 **숫자 입력란**과 양 옆의 **`−`**, **`+`** 버튼이 있습니다. 커브 전체를 정수 단위로 위/아래로 이동시킵니다. 더 세밀한 조정이 필요하다면 숫자 입력란에 소수점을 입력할 수도 있습니다.

세로 축 위치 설정은 다음과 같은 상황에 유용합니다:

- 음색 균형을 비교하지만 두 헤드폰의 음압 차이가 크게 발생할 때 — 하나의 커브를 상하로 조절하여 중음역대를 맞춘 후 전체적인 선형 _형태_를 비교할 수 있습니다.
- 여러 개의 커브가 서로 겹치지 않도록 시각적으로 분산 배치하고 싶을 때.

## 타겟 커스터마이저 (타겟 전용)

타겟 커브에는 일반 헤드폰에 없는 기능이 추가됩니다. **타겟 커스터마이저** 버튼(슬라이더 아이콘)입니다. 클릭하면 기울기 설정, 저음, 고음, 이어게인 조절기 및 원클릭 프리셋이 포함된 화면이 열립니다. 사이트 운영자가 맞춤형 설정을 허용한 타겟 파일에만 이 버튼이 표시됩니다. 작업 방식은 [타겟과 선호도](./09-targets-and-preferences.mdx) 문서를 참고하십시오.
