# 개요

modernGraphTool은 헤드폰, IEM, 스피커의 주파수 응답(FR) 측정값을 시각화하고 비교하기 위한 웹 기반 도구입니다. 측정값을 불러오고, 여러 커브를 겹치고, 타겟과 EQ를 적용하고, 결과를 공유하는 모든 작업을 브라우저에서 수행할 수 있습니다.

SvelteKit 2, Svelte 5, Tailwind CSS, D3.js로 구축되었습니다. 완전한 정적 SPA로 배포되며, `dist/` 폴더를 아무 웹 서버에 업로드하거나 자동 업데이트를 원한다면 CDN 모드를 사용할 수 있습니다.

modernGraphTool v1에서 넘어오시는 경우 변경 사항은 [마이그레이션 가이드](./migrating-v1-to-v2)를 참고하세요. 이 프로젝트가 존재하는 이유에 대한 배경은 [왜 modernGraphTool인가?](./why-moderngraphtool)에서 확인할 수 있습니다.

## 기능

모든 기능은 기본 제공되며 단일 [`config.js`](./guide-for-admins/customize-page) 파일을 통해 설정할 수 있습니다. 개별 기능에 대한 자세한 문서는 [기능](./category/features) 섹션을 참고하세요.

### 시각화 및 분석

- **주파수 응답 그래프** — D3.js로 구동되는 SVG 그래프, 로그 스케일 주파수 축 (20 Hz – 20 kHz), 설정 가능한 Y축 스케일 (30–80 dB 범위)
- **다중 커브 비교** — 여러 기기를 불러와 겹쳐 비교하고, 커브별로 색상 커스터마이즈 가능
- **타겟 커브** — 내장 타겟 (Harman, DF, 리뷰어 선호도)을 선택하거나 직접 추가할 수 있으며, 설정 가능한 그룹으로 정리됩니다.
- **정규화** — 특정 주파수(Hz) 기준 또는 중역대 평균(300 Hz – 3 kHz) 기준으로 커브를 맞춥니다.
- **스무딩** — 1/48 옥타브 (거의 원본)부터 1/3 옥타브까지 조절 가능
- **다중 샘플 측정** — 기기에 여러 측정 실행이 있을 때 개별 샘플 또는 평균을 표시합니다.
- **HpTF 편차** — 헤드폰 전달 함수의 샘플 간 편차를 음영 영역, 개별 커브, 또는 둘 다로 시각화합니다.
- **Inspection 모드** — 그래프 위에 마우스를 올려 정확한 주파수와 dB 값을 읽을 수 있습니다.
- **기준선 보정** — 임의 커브를 기준선으로 설정해 다른 커브를 상대 차이로 볼 수 있습니다.
- **그래프 종횡비 설정** — 표준 16:9 또는 CrinGraph 스타일 비율

### 이퀄라이저

- **[파라메트릭 EQ](./features/equalizer)** — 다양한 필터 타입 (PEQ, LSQ, HSQ), 조절 가능한 주파수/게인/Q, 프리앰프 제어가 포함된 풀 기능 EQ 패널
- **AutoEQ** — 선택한 타겟 커브에 맞춰 EQ 프로필을 자동 생성
- **실시간 오디오 미리 듣기** — EQ 필터가 실시간으로 적용된 오디오를 브라우저에서 재생
- **가져오기 / 내보내기** — EQ 설정을 저장하고 나중에 불러올 수 있습니다.
- **[디바이스 PEQ 브리지](./features/device-peq)** — USB HID, Serial, BLE, 또는 네트워크를 통해 20여종의 호환 하드웨어 기기에 EQ 설정을 직접 전송

### 타겟 커스터마이징

- **[Target Customizer](./features/target-customizer)** — 슬라이더로 파라메트릭 필터 (Tilt, Bass, Treble, Ear Gain 등)를 사용해 타겟 커브를 조정
- **필터 프리셋** — 인기 튜닝 프로파일 (Harman 2013, 2015, 2018)을 원클릭으로 적용
- **타겟별 초기 필터** — 타겟이 로드될 때 선호 조정을 자동으로 적용
- **[Preference Bound](./features/preference-bound)** — 연구 데이터를 기반으로 한 음영 처리된 상한/하한 선호 범위를 그래프에 오버레이

### 공유 및 내보내기

- **URL 공유** — 전체 그래프 상태 (로드된 기기, 타겟, 정규화, EQ)를 인코딩하는 압축된 공유 가능 링크 생성
- **스크린샷** — 현재 그래프를 라벨과 워터마크가 포함된 이미지로 저장
- **워터마크** — 위치와 투명도를 조절할 수 있는 텍스트 또는 이미지 워터마크

### 사용자 인터페이스

- **반응형 레이아웃** — 작은 화면에 억지로 끼워 맞춘 데스크톱 뷰가 아닌, 전용 데스크톱/모바일 UI
- **다크 모드** — 라이트, 다크, 또는 시스템 설정 따르기
- **패널 내비게이션** — 기능을 Device, Graph, Equalizer, Misc 네 개 패널로 구성
- **조정 가능한 레이아웃** — 데스크톱에서 구분선을 드래그해 그래프와 패널 영역의 크기를 조절
- **다국어** — 컴파일 타임 i18n (Paraglide JS)로 영어와 한국어 지원
- **설정 가능한 상단 바** — 커스텀 제목 (텍스트, 이미지, HTML), 내비게이션 링크, 브랜딩
- **[Frequency Tutorial](./features/frequency-tutorial)** — 초심자에게 유용한, 주파수 대역의 이름과 설명을 보여주는 인터랙티브 오버레이

### squig.link 연동

`*.squig.link` 도메인에서 호스팅될 때, modernGraphTool은 [squig.link](https://squig.link) 네트워크를 위한 추가 기능들을 활성화합니다. 자세한 내용은 [squig.link 연동](./features/squiglink-integration)을 참고하세요.

- **크로스 사이트 검색** — squig.link 네트워크의 모든 사이트에서 기기를 검색
- **쇼핑 링크** — 샵 데이터베이스에서 일치하는 기기를 찾았을 때 구매 링크 표시
- **분석** — 네트워크 전체 추적을 위한 다중 측정 ID 지원 Google Analytics
- **스폰서 콘텐츠** — 참여한 사이트의 첫 방문 스폰서 배너
