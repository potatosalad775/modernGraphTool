# 커브 다루기

불러온 커브는 그래프 패널 안의 **선택 목록**에 한 줄씩 나타납니다. 특정 커브를 조절하려면 그래프 패널(두 번째 메뉴 아이콘, 데스크톱에서는 **2** 키)로 전환하십시오. 각 행의 레이아웃은 데스크톱과 모바일이 같습니다.

{/* TODO: screenshot of a selection-list row with the control cluster annotated */}

이 문서는 선택 기기 한 항목에 딸린 모든 조작 요소를 좌측 상단에서 우측 하단 순서로 설명합니다. 선택 목록 안에서도 타겟은 맨 위에 정렬되어 찾기 쉽습니다.

## 선택 기기 레이아웃

선택 기기 한 항목은 두 줄짜리 레이아웃입니다.

- **상단 레이아웃:** 색상 버튼, 기기 이름(상세 항목이 있으면 화살표 표식 포함), 우측에 모인 작업 버튼.
- **하단 레이아웃:** 채널 선택 버튼과 세로축 위치 제어 버튼.

## 색상 버튼

좌측의 작은 사각형은 그래프에 그려진 해당 기기의 **색상**입니다. 이 버튼을 클릭하면 [색상 선택기](./12-appearance-and-language.mdx#per-curve-color)가 열립니다. 색상 선택 칸, HSL 입력란, 점선 패턴 제어, 무작위 색상 버튼이 같이 나옵니다. 여러 커브의 색이 비슷해서 한 커브에만 다른 색을 주고 싶을 때 씁니다.

새 그래프가 추가될 때마다 자동으로 다른 색이 지정되니, 실제로 손댈 일은 많지 않습니다.

## 이름 및 상세 항목 선택란

측정 파일이 하나뿐인 기기는 이름만 표시됩니다. 반대로 이어팁, 이어패드, 착용 위치 등 **상세 항목**이 여럿인 기기는 이름 옆에 작은 화살표 기호가 붙습니다. 이름을 클릭하면 상세 항목이 모두 나열된 목록이 열리고, 원하는 항목을 고르면 현재 그래프의 커브가 그 항목으로 **교체**됩니다.

두 항목을 동시에 그래프에 띄우고 싶다면, 원하는 항목 오른쪽의 **플러스 버튼**을 누르십시오.

상세 항목은 사이트 측정 데이터베이스에서 기기마다 따로 구성됩니다. 상세 항목 데이터가 없는 기기에는 화살표 표식이 나오지 않습니다.

## 기준선

눈 아이콘 옆 파형 모양의 **기준선** 버튼은 해당 커브를 기준점으로 고정하고 _다른_ 커브를 모두 이 기준점과의 차이 값으로 보여줍니다. "FiiO 이어폰과 하만 타겟은 3 kHz 대역에서 어떻게 다른가?" 같은 의문을 풀 때 쓸 만합니다. 두 선을 수직으로 눈대중 비교하지 않고, 타겟을 기준선으로 잡아 그래프에서 차이를 곧장 읽어냅니다.

기준선을 어디에 거느냐에 따라 작동 방식이 두 가지입니다:

**일반 헤드폰, 또는 사용자 맞춤화가 적용되지 않은 타겟:** 단순한 켜기/끄기 토글입니다. 한 번 누르면 켜지고, 다시 누르면 꺼집니다.

**선호도 조정 타겟** ([타겟 커스터마이저](./09-targets-and-preferences.mdx) 사용): 버튼이 세 가지 상태를 순환합니다. 기기 비교 분석과 분리해서, 사용자가 직접 손본 튜닝 타겟의 효과만 따로 확인할 수 있습니다.

| 상태 | 의미 |
|---|---|
| **꺼짐** | 일반 그래프 모드로, 기준선이 없습니다. |
| **조정 전** | 선호도 조정 이전의 _원본_ 타겟을 기준선으로 설정합니다. |
| **조정 후** | 선호도 조정값을 _포함한_ 타겟을 기준선으로 설정합니다. |

버튼 위에 잠시 커서를 올리면, 해당 커브의 현재 상태를 텍스트로 알려줍니다. 클릭하면 다음 상태로 넘어갑니다.

:::info[기준선은 데이터 변환이 아닌 시각 보조 도구입니다]
기준선을 켜도 기본 측정 파일이나 EQ 파이프라인은 바뀌지 않습니다. 기존 커브가 그려지는 위치 좌표만 달라집니다. 기능을 끄면 원래 화면이 다시 나옵니다.
:::

## 표시 화면

**눈 아이콘**은 목록에서 커브를 지우지 않고 그래프 영역에서만 잠시 숨기는 역할을 합니다. 클릭 시 아이콘이 바뀜과 동시에 배경이 어두워지는 것으로 숨김 상태임을 알립니다. 다시 누르면 화면에 나타납니다. 커브를 불러둔 채로 "_이 커브_를 빼면 그래프가 어떻게 보일까"를 확인하고 싶을 때 씁니다.

## 삭제

**휴지통 아이콘**은 커브를 완전히 지웁니다. **Ctrl+Z** 단축키로 직전 작업을 되돌릴 수 있어, 실수로 클릭했더라도 쉽게 이전 상태를 복구할 수 있습니다.

## 채널 선택기

하단 레이아웃에는 채널 선택기가 놓입니다(타겟 그래프에는 나타나지 않습니다). 측정 파일에 따라 옵션이 달라집니다:

- **L** — 왼쪽 채널 곡선만 그림
- **R** — 오른쪽 채널 곡선만 그림
- **L+R** — 두 곡선을 따로 그림
- **AVG** — 양쪽 채널을 평균 내 한 선으로 그림

`L+R` 옵션이 제품 특성을 가장 자세히 보여주고, `AVG` 옵션이 가장 간결합니다. 측정 데이터가 단일 채널뿐이면 옵션도 하나만 나옵니다.

### 샘플 선택기 (다중 샘플 기기용)

같은 모델을 여러 번 측정한 **다중 샘플** 데이터가 있는 기기라면, 더 다양한 설정이 담긴 **샘플/채널 선택기**가 기존 선택기를 대신하며, 다음 기능이 추가됩니다.

- 특정 샘플 지정, 또는 전체 샘플 평균 표시.
- 머리 전달 함수 편차를 **음영 대역**, **개별 커브**, 혹은 둘 다로 표시할지 선택.

이 기능으로 "이 헤드폰이 일관된 성능을 내는가, 개체 간 편차가 큰가"를 가늠합니다. 그래프 대역이 좁으면 편차가 적고, 넓으면 개체 간 특성 차이가 큽니다.

## 세로 축 위치 제어

채널 선택기 우측에는 **숫자 입력란**과 양옆의 **`−`**, **`+`** 버튼이 있습니다. 커브 전체를 정수 단위로 위·아래로 옮깁니다. 더 세밀하게 맞추려면 숫자 입력란에 소수점도 입력합니다.

세로 축 위치 설정은 다음 상황에서 쓸모 있습니다:

- 음색 균형은 비교하고 싶은데 두 헤드폰의 음압 차이가 클 때 — 한 커브를 위아래로 조절해 중음역대를 맞춘 뒤 전체 선의 _형태_를 비교합니다.
- 여러 커브가 서로 겹치지 않도록 시각적으로 분산 배치하고 싶을 때.

## 타겟 커스터마이저 (타겟 전용)

타겟 커브에는 일반 헤드폰에 없는 기능이 하나 더 붙습니다. **타겟 커스터마이저** 버튼(슬라이더 아이콘)입니다. 클릭하면 기울기 설정, 저음, 고음, 이어게인 조절기와 원클릭 프리셋이 담긴 화면이 열립니다. 사이트 운영자가 맞춤형 설정을 허용한 타겟 파일에만 이 버튼이 나옵니다. 작업 방식은 [타겟과 선호도](./09-targets-and-preferences.mdx) 문서를 참고하십시오.
