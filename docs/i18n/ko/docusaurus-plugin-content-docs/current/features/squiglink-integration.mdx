# squig.link Integration

modernGraphTool을 squig.link 서비스와 연결하는 내장 통합 기능입니다.

## 개요

squig.link Integration은 squig.link 생태계와의 원활한 통합을 제공하여 modernGraphTool을 향상시킵니다. 사이트 간 검색, 쇼핑 링크, 분석 추적, 스폰서 배너 기능을 지원합니다.

:::note
squig.link 기능은 modernGraphTool이 *.squig.link 도메인에서 호스팅될 때만 활성화됩니다.
:::

## 주요 기능

- **사이트 간 검색**: squig.link 네트워크의 모든 사이트에서 기기 검색
- **쇼핑 링크**: 매칭되는 쇼핑 항목이 있는 기기에 "Buy Now" 링크 표시
- **분석**: 다중 측정 ID를 지원하는 Google Analytics 통합
- **스폰서 배너**: 첫 방문 시 스폰서 콘텐츠 표시
- **설정 커스터마이즈**: 다양한 배포 환경에 맞는 유연한 설정

## 설정

squig.link Integration은 `config.js`에서 설정합니다:

```javascript
// In config.js
SQUIGLINK: {
    ENABLED: true,
    DEBUG: false,
    ANALYTICS_MEASUREMENT_IDS: [],
    ANALYTICS_SITE: "",
    LOG_ANALYTICS: true,
    ENABLE_ANALYTICS: true,
    ENABLE_CROSS_SITE_SEARCH: true,
    ENABLE_SPONSOR: true,
}
```

### 설정 옵션

- **`ENABLED`**: 모든 squig.link 기능의 마스터 토글
- **`DEBUG`**: 문제 해결을 위한 디버그 로깅 활성화
- **`ANALYTICS_MEASUREMENT_IDS`**: 다중 태그 지원을 위한 Google Analytics 4 측정 ID 배열
- **`ANALYTICS_SITE`**: 분석 이벤트 귀속에 사용되는 사이트 이름
- **`LOG_ANALYTICS`**: 디버깅을 위해 분석 이벤트를 콘솔에 로깅
- **`ENABLE_ANALYTICS`**: 분석 추적 활성화/비활성화 토글
- **`ENABLE_CROSS_SITE_SEARCH`**: 사이트 간 기기 검색 토글
- **`ENABLE_SPONSOR`**: 스폰서 배너 표시 토글

## 사용법

### 사이트 간 검색
참여 중인 모든 squig.link 사이트에서 헤드폰 측정값을 검색합니다. 검색 결과는 해당 사이트의 그래프 도구로 직접 연결되며 기기가 로드된 상태로 표시됩니다.

### 쇼핑 링크
로드된 기기에 squig.link 쇼핑 링크 데이터베이스의 매칭 항목이 있을 경우, 판매처로 연결되는 "Buy Now" 버튼이 표시됩니다.

### 분석
활성화 시, 사이트 사용 인사이트를 위해 Google Analytics 4 (gtag.js)를 통해 사용자 상호작용이 추적됩니다.

### 스폰서 배너
첫 방문 시, squig.link 네트워크에서 로드된 콘텐츠가 포함된 스폰서 다이얼로그가 표시될 수 있습니다.
