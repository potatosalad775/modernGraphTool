# 이 도구는 무엇인가요?

modernGraphTool은 헤드폰 소리를 귀가 아닌 그래프로 확인하고 비교하는 웹 기반 분석 도구입니다.

헤드폰을 측정할 때는 사람이 들을 수 있는 전체 주파수 대역(극저음부터 초고음까지)을 담은 테스트 신호를 재생합니다. 이때 측정용 더미 헤드에 내장된 마이크가 각 대역의 음압 변화를 기록하고, 이를 선으로 이은 결과물이 바로 **주파수 응답(Frequency Response) 커브**입니다. modernGraphTool은 이렇게 측정된 데이터를 불러와 서로 겹쳐 놓고 한눈에 비교하도록 도와줍니다.

화면에 표시되는 커브는 대부분 측정 마이크가 기록한 소리를 그대로 옮긴 **raw(원본)** 데이터입니다. 즉 "헤드폰이 순수하게 내는 소리"만 분리한 것이 아니라, 측정 장비 고유의 물리적 음향 특성까지 한데 섞인 결과물입니다.

그래서 이 형태를 제대로 해석하려면 반드시 **타겟(Target)** 커브를 함께 봐야 합니다. 타겟은 특정 측정 장비의 특성을 고려해 고안된 기준선입니다. 이 기준선은 완벽히 중립적인 음색을 목표로 삼기도 하고, 하만(Harman) 타겟처럼 다수의 연구 데이터가 증명하는 대중적 선호도를 반영하기도 하며, 리뷰어 개인의 취향을 묘사하기도 합니다.

raw 커브가 타겟 라인에 가까울수록 해당 타겟이 추구하는 소리에 부합하고, 두 선이 어긋나는 지점이 바로 헤드폰 고유의 튜닝 편차입니다. 이 개념은 이어지는 03, 04번 문서에서 더 자세히 다루므로, 여기서는 **"화면에 보이는 선은 원본 데이터(raw)이고, 이를 바르게 읽기 위한 해석 기준선이 타겟"**이라는 점만 기억하시면 충분합니다.

## 이 도구로 할 수 있는 것

- **헤드폰 비교:** 여러 기기를 동시에 불러와 주파수 응답 차이를 한눈에 살펴봅니다. 예를 들어 저음이 강한 헤드폰은 그래프 왼쪽 영역이 눈에 띄게 솟아오릅니다.
- **타겟 오버레이:** 각 측정 장비별로 '이상적인 소리'가 어떻게 측정되어야 하는지 보여주는 기준선입니다. 음향적 중립, 대중적 선호도, 특정 리뷰어의 취향 등 다양한 타겟을 겹쳐 비교합니다.
- **EQ 시뮬레이션:** 특정 주파수 대역의 피크(Peak)가 거슬린다면, 브라우저에서 즉시 해당 대역의 게인을 낮춰 오디오 샘플을 재생하고 변화를 직접 들어봅니다.
- **결과 공유:** 작업 중인 그래프의 모든 커스터마이징 상태가 URL에 그대로 인코딩됩니다. 짧은 링크 하나만 전달해도 다른 사람과 동일한 화면을 손쉽게 공유합니다.

## 음향 전문가를 위한 도구인가요?

그렇지 않습니다. 오디오 리뷰어나 매니아들이 널리 활용하는 도구이긴 하지만, 복잡한 마케팅 수사 대신 헤드폰의 실제 성향을 있는 그대로 파악하고 싶은 일반 사용자도 충분히 다룹니다. 음향 공학에 대한 깊은 지식은 필요하지 않습니다.

주파수 응답 그래프가 처음이라면 이어지는 세 편의 문서를 가벼운 입문서로 활용해 보세요. 그래프가 무엇을 보여주는지, 왜 굴곡진 선들이 나타나는지, 사람들이 말하는 "플랫(Flat)한 소리"가 왜 생각보다 복잡한 개념인지 차근차근 설명해 드립니다. 입문 문서들을 읽고 나면 도구의 각 기능과 조작법을 다루는 실전 가이드도 한결 수월하게 따라오실 수 있습니다.
