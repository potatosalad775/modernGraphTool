# 공유 및 내보내기

원하는 내용을 전달할 수 있는 그래프 뷰를 완성했다면, 누군가에게 보여주고 싶을 것입니다. modernGraphTool은 이를 위해 크게 세 가지 기능을 제공합니다. 그래프 도구 모음의 **공유**, **스크린샷**, 그리고 이퀄라이저 패널 내부의 EQ 파일 **가져오기/내보내기**입니다.

## URL 공유

그래프 도구 모음에서 **공유** 버튼을 클릭하십시오. 이 도구는 현재 화면의 간결하게 인코딩된 스냅샷이 포함된 URL을 클립보드에 복사합니다. 여기에는 다음 정보가 포함됩니다.

- 로드된 모든 기기 및 타겟
- 세로축 위치 설정, 기기 세부 항목, 채널 선택, 색상
- 정규화 모드, 스무딩, Y축 스케일, 기준선
- 타겟 커스터마이저 조정 값
- 활성화된 경우 EQ 필터 목록

채팅 메시지, 포럼 게시글, 리뷰어 기사, 버그 리포트 등 원하는 곳 어디에든 URL을 붙여넣으십시오. 다른 사람이 브라우저에서 이 URL을 열면 modernGraphTool은 정확히 동일한 기기 구성으로 똑같은 화면을 재구성합니다.

:::info[계정 및 서버 불필요]
공유 URL은 어떠한 데이터도 업로드하지 않습니다. 모든 상태량은 주소 표시줄 자체에 인라인 형태로 인코딩됩니다. 단축 링크 서비스나 공유 데이터를 저장하는 데이터베이스, 사용량 제한이 존재하지 않습니다. 본인에게 주소가 작동한다면 받은 사람 누구에게나 작동합니다(단, 잠시 후 설명하겠지만 두 사람이 동일한 측정 사이트에 접속해야 합니다).
:::

### 공유 URL이 길고 복잡해 보이는 이유

URL에는 모든 상태를 나타내는 base62 인코딩 청크가 포함되어 있습니다. 의도적으로 복잡한 형태를 띠며, 이 청크 자체가 핵심입니다. 짧은 주소를 선호한다면 게시하기 전에 URL 단축기를 사용하십시오.

### 타 사이트 간의 주의 사항

공유 URL은 _현재 접속 중인 사이트_ 기준으로 생성됩니다. _site-A.squig.link_에서 화면을 구성하고 URL을 복사하여 다른 사람이 _site-B.squig.link_에서 열게 되면, B 사이트는 _자신의_ 데이터베이스에서 동일한 기기 이름을 로드하려고 시도하기 때문에 B 사이트에 존재하지 않는 이름은 조용히 무시됩니다.

수신자가 본인이 본 화면을 정확히 보길 원한다면 동일한 사이트에서 URL을 열도록 안내하십시오. 타 사이트 간 링크는 두 사이트가 우연히 동일한 기기 이름을 제공하는 경우에만 작동합니다.

## 스크린샷

**스크린샷** 버튼을 클릭하면 modernGraphTool이 축 레이블 등을 포함한 현재 그래프 영역의 PNG 이미지를 생성하여 브라우저의 기본 다운로드 경로로 다운로드합니다.

스크린샷에는 다음이 포함됩니다.

- 표시된 모든 커브가 있는 그래프
- 축 레이블
- 사이트 운영자가 구성한 워터마크 (사이트의 config.js를 통해 텍스트나 이미지로 지정 및 스타일링됨)
- 로드된 경우 타겟 커브

패널, 도구 모음 또는 그래프 영역 밖의 다른 모든 요소는 포함되지 _않습니다_. 전체 페이지 캡처가 아닌 그래프 전용 내보내기 기능입니다.

다음과 같은 상황에 스크린샷을 사용하는 것이 좋습니다.

- 리뷰나 기사를 작성하며 정적 이미지를 삽입하고자 할 때
- 공유 URL을 사용하기 어려운 장치(출력물, 슬라이드 덱, PDF 등)에서 열람할 때
- 기본 데이터베이스가 변경되더라도 깨지지 않는 영구적인 기록을 남기고자 할 때

정적 이미지보다 실시간 링크를 선호한다면 공유 기능을 사용하십시오. 수신자는 모든 제어 기능이 여전히 작동하는 대화형 버전을 얻을 수 있습니다.

## 이퀄라이저 내보내기

이퀄라이저 패널에는 EQ 필터 목록을 위한 고유한 가져오기 및 내보내기 기능이 있습니다. 이는 공유 URL 시스템과는 별개인 일반 텍스트 파일입니다. 다른 파일과 마찬가지로 저장하거나 이메일로 전송하거나 다른 EQ 애플리케이션으로 로드할 수 있습니다.

작업 흐름에 대한 내용은 [오디오 이퀄라이징](./10-equalizing.mdx) 및 [이퀄라이저](../features/equalizer)를 참조하십시오.

## 실행 취소 및 다시 실행

"작업 내역 손실 방지"와 관련하여 덧붙이자면, 기기를 로드하거나 제거하거나 또는 개별 커브 설정을 변경하는 작업은 **실행 취소 스택**에 저장됩니다. **Ctrl+Z**를 누르면 실행을 취소하고, **Ctrl+Shift+Z**(또는 **Ctrl+Y**)를 누르면 다시 실행합니다. 조정하는 데 공을 들인 커브를 실수로 삭제했더라도 단축키 한 번으로 복구할 수 있습니다.

실행 취소 기능이 _모든_ 상태 변경에 적용되는 것은 아닙니다. 스무딩이나 Y축 스케일과 같은 도구 모음 뷰 설정은 실행 취소 대상이 아니지만, 로드 및 제거, Y축 오프셋, 커스터마이저 조정과 같이 가장 중요한 항목들은 지원됩니다.
