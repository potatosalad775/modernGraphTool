# Frequency Tutorial

modernGraphTool에서 오디오 주파수 특성을 이해할 수 있도록 주파수 범위 튜토리얼을 제공하는 내장 기능입니다.

## 개요

Frequency Tutorial은 modernGraphTool에 교육적 요소를 추가하여, 사용자가 다양한 주파수 범위와 그 오디오 특성을 이해할 수 있도록 도와줍니다.

각 주파수 대역과 그에 해당하는 일반적인 오디오 콘텐츠에 대한 시각적, 텍스트 정보를 제공합니다.

## 주요 기능

- **인터랙티브 주파수 가이드**: 주파수 범위의 시각적 표현
- **다국어 지원**: 언어별 콘텐츠를 통한 국제화 지원
- **시각적 통합**: modernGraphTool의 그래프 인터페이스와 완벽하게 통합

## 사용법

Frequency Tutorial은 그래프 위에 주파수 대역 정보를 오버레이하여, 서브베이스, 베이스, 중음역, 트레블 등의 주파수 범위를 레이블과 함께 표시하고 각 오디오 특성에 대한 설명을 제공합니다.

### 튜토리얼 내용 편집

`static/assets/strings/`에 있는 문자열 파일을 편집하여 각 주파수 범위의 튜토리얼 설명을 커스터마이즈할 수 있습니다.

```json
[
  {
    "name": "Sub Bass",
    "range": [20, 60],
    "description": "The Rumble, usually out of human's hearing range and tend to be felt more than heard, providing a sense of power."
  },
  {
    "name": "Bass",
    "range": [60, 250],
    "description": "Determins how 'fat' or 'thin' the sound is, boosting around 250hz tend to add a feeling of warmth. If you're a bass head you most likely like this range."
  },
  { ... More Ranges ... }
]
```
