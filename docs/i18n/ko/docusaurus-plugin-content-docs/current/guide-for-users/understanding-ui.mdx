# 화면 영역 이해하기

![UI screenshot with color overlay](./img/guide-user-ui-ko-colored.png)

modernGraphTool의 사용자 인터페이스는 네비게이션 바를 통해 접근할 수 있는 4개의 패널로 구성되어 있습니다:

- **Device 패널**: 브랜드별 기기 검색 및 탐색, 비교를 위한 다중 기기 불러오기, 변형(variant) 및 채널 선택.
- **Graph 패널**: 타겟 선택, 정규화, 스무딩, Y축 스케일, 스크린샷, 공유 URL, 주파수 튜토리얼, 선호 범위(Preference Bounds), 그래프 색상 커스터마이징.
- **Equalizer 패널**: AutoEQ를 활용한 파라메트릭 EQ, 실시간 오디오 재생, 필터 관리, 기기 PEQ 브릿지, 타겟 커스터마이저.
- **Misc 패널**: 다크 모드 토글, 언어 선택, 설명 영역, 후원 버튼.

## 데스크탑 UI

1000px 이상의 화면 너비를 가진 기기에서는 '데스크탑 UI'가 적용됩니다.
- '그래프 영역'이 좌측에, 패널 영역과 '네비게이션 바'가 우측에 배치됩니다.
- '그래프 영역'과 패널 영역 사이를 나누는 수직 구분선을 좌우로 드래그하여 두 영역 사이의 비율을 조절할 수 있습니다.
- 현재 그래프에 표시되고 있는 데이터 목록을 나열하는 도구가 '그래프 영역'의 하단으로 이동합니다.

## 모바일 UI

반면, 좁은 화면 너비를 가진 모바일 기기에서는 '모바일 UI'가 적용됩니다.
- '그래프 영역' 아래에 패널 영역과 '네비게이션 바'가 차례로 배치됩니다.
- '그래프 영역'과 패널 영역 사이의 비율을 조절할 수 없습니다.
- 현재 그래프에 표시되고 있는 데이터 목록을 나열하는 도구가 패널 영역으로 병합됩니다.
