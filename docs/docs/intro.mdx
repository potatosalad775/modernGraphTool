# Overview

modernGraphTool is a web-based Single Page Application for visualizing and analyzing frequency response (FR) data for audio devices. It is built with SvelteKit 2 and Svelte 5, rewritten from the ground up as a v2 successor to the original vanilla JavaScript version.

If you're coming from modernGraphTool v1, see the [migration guide](./migrating-v1-to-v2) for what's changed.

To learn about the motivation behind the project, see [Why modernGraphTool?](./why-moderngraphtool).

## Built-in Features

All features are included out of the box and configurable through a single `config.js` file. For detailed documentation on each, see the [Features](./category/features) section.

| Name                   | Description |
|------------------------|-------------|
| Equalizer              | Parametric EQ panel with AutoEQ and real-time audio playback |
| Device PEQ             | Hardware EQ bridge supporting USB HID, Serial, BLE, and Network devices |
| Target Customizer      | Target curve customization with Tilt, Bass, Treble presets |
| Preference Bound       | Customizable preference bound overlay on the graph |
| Frequency Tutorial     | Interactive frequency range guide for beginners |
| squig.link Integration | Cross-site search, shop links, and analytics for squig.link network |

## Graph Tools

- Share with URL (Base62 compressed)
- Save frequency response graph as PNG screenshot
- Normalization — Hz-based or Midrange Average (300Hz~3kHz)
- Smoothing — 1/48 oct to 1/3 oct
- Multi-sample measurement display (average or individual)
- HpTF (Headphone Transfer Function) deviation display
- Inspection mode for detailed measurement values
- Graph color and appearance customization per curve

## User Interface

- Responsive layout for desktop and mobile
- Dark mode support (light, dark, or system preference)
- Multi-language support (English, Korean)
- Configurable top bar, watermark, description, and graph theme
