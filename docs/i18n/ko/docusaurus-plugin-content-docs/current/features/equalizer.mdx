# Equalizer

modernGraphTool에 내장된 본격 파라메트릭 이퀄라이저로, 실시간 오디오 EQ를 제공합니다.

## 개요

Equalizer는 modernGraphTool에 내장된 패널로, 완전한 파라메트릭 이퀄라이저 기능을 제공합니다. 헤드폰 측정값에 실시간으로 오디오 필터를 적용해 결과를 바로 들어볼 수 있습니다.

다양한 필터 타입, 프리앰프 제어, EQ가 적용된 오디오 재생까지 모두 지원합니다.

## 주요 기능

- **파라메트릭 EQ** — Peaking, Low Shelf, High Shelf 등 여러 필터 타입과 주파수·게인·Q 조정
- **프리앰프 자동 계산** — EQ 적용 시 클리핑을 막기 위해 프리앰프 게인을 자동으로 계산
- **실시간 오디오** — 내장 오디오 플레이어에 EQ 필터를 실시간으로 적용
- **필터 관리** — EQ 밴드를 동적으로 추가·삭제·수정
- **헤드폰 선택** — 여러 헤드폰 측정값 사이를 빠르게 전환
- **AutoEQ** — 타겟 커브에 맞춘 EQ 프로파일 자동 생성. Low/High Shelf 필터 옵션도 함께 지원
- **가져오기/내보내기** — EQ 설정을 저장하거나 불러오기
- **파일 업로드** — 커스텀 주파수 응답·타겟 파일 업로드

## 사용법

1. **기본 EQ** — 필터 컨트롤로 주파수, 게인, Q 값을 조정합니다.
2. **헤드폰 선택** — 헤드폰 선택기로 다른 측정값으로 전환합니다.
3. **오디오 재생** — 오디오 재생을 켜면 EQ 변화를 실시간으로 들을 수 있습니다.
4. **AutoEQ** — 선택한 타겟 커브에 맞춰 EQ 프로파일을 자동 생성합니다.
5. **가져오기/내보내기** — EQ 설정을 파일로 저장하거나 다시 불러옵니다.

## 참고 사항

- 오디오 기능을 사용하려면 Web Audio API를 지원하는 최신 브라우저가 필요합니다.
- 실시간 오디오 처리는 CPU 사용량을 늘릴 수 있습니다.
- EQ 밴드를 너무 많이 쓰면 구형 기기에서 성능이 떨어질 수 있습니다.
