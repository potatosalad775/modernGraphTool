# 개요

modernGraphTool은 헤드폰과 이어폰의 주파수 응답(FR) 측정 데이터를 시각화하고 비교하는 웹 도구입니다. 측정 데이터 불러오기, 여러 그래프 겹치기, 타겟·EQ 적용, 결과 공유까지 모두 브라우저에서 바로 처리합니다.

SvelteKit 2, Svelte 5, Tailwind CSS, D3.js로 만든 정적 SPA로, `dist/` 폴더를 어떤 웹 서버에 올려도 동작합니다. CDN 모드를 쓰면 자동 업데이트도 함께 받습니다.

modernGraphTool v1에서 넘어오신다면 [전환 가이드](./migrating-v1-to-v2)를 참고하세요. 프로젝트의 배경이 궁금하다면 [왜 modernGraphTool인가?](./why-moderngraphtool) 페이지를 보시면 됩니다.

## 기능

기본 기능은 모두 내장되어 있고, 단일 [`config.js`](./guide-for-admins/customize-page) 파일로 설정합니다.

각 기능에 대한 자세한 설명은 [기능](./category/features) 섹션을 참고하세요.

### 시각화 및 분석

- **주파수 응답 그래프** — D3.js 기반 SVG 그래프, 로그 스케일 주파수 축(20 Hz – 20 kHz)과 사용자 설정 Y축 스케일(30–80 dB)
- **다중 곡선 비교** — 여러 기기를 동시에 불러와 겹쳐 보고, 곡선마다 색상을 지정
- **타겟 커브** — 내장 타겟(Harman, Diffuse Field, 리뷰어 선호도) 중에서 고르거나 직접 추가, 그룹으로 묶어 정리
- **정규화** — 특정 주파수(Hz) 또는 중역대 평균(300 Hz – 3 kHz)을 기준으로 곡선을 정렬
- **스무딩** — 원본에 가까운 1/48 옥타브부터 1/3 옥타브까지 단계 조정
- **다중 샘플 측정** — 한 기기를 여러 번 측정한 경우, 개별 샘플 또는 평균값을 골라 표시
- **HpTF 편차 범위** — 헤드폰 전달 함수의 샘플 간 편차를 음영, 개별 곡선, 또는 둘 다로 시각화
- **조사 모드** — 그래프 위에 마우스를 올려 정확한 주파수와 dB 값을 확인
- **기준선 보정** — 특정 곡선을 기준선으로 잡아 다른 곡선을 상대 차이로 표시
- **종횡비 설정** — 표준 16:9 또는 CrinGraph 스타일 비율 지원

### 이퀄라이저

- **[파라메트릭 EQ](./features/equalizer)** — 다양한 필터 타입(PEQ, LSQ, HSQ), 주파수·게인·Q 조정, 프리앰프 제어를 갖춘 본격 EQ 패널
- **AutoEQ** — 선택한 타겟 커브에 맞춰 EQ 프로파일을 자동 생성. LSQ/HSQ 필터 옵션이 추가되어 튜닝 자유도가 넓어졌습니다.
- **실시간 오디오 미리 듣기** — EQ 필터가 적용된 음악을 브라우저에서 바로 들어봅니다.
- **가져오기 / 내보내기** — EQ 설정을 텍스트 파일로 저장하거나 불러옵니다.
- **[Device PEQ 브리지](./features/device-peq)** — USB HID, Serial, BLE, 네트워크로 20여 종의 호환 하드웨어에 EQ 설정을 직접 전송

### 타겟 커스터마이징

- **[Target Customizer](./features/target-customizer)** — Tilt, Bass, Treble, Ear Gain 등 파라메트릭 필터를 슬라이더로 조정해 타겟 커브를 다듬습니다.
- **필터 프리셋** — 인기 튜닝 프로파일(Harman 2013, 2015, 2018)을 클릭 한 번에 적용
- **타겟별 초기 필터** — 타겟이 로드될 때 미리 정해 둔 조정값을 자동 적용
- **[선호도 경계](./features/preference-bound)** — 연구 데이터를 바탕으로 한 상·하한 선호 범위를 그래프 위에 음영으로 표시

### 공유 및 내보내기

- **URL 공유** — 그래프 상태(로드된 기기, 타겟, 정규화, EQ)를 통째로 담은 압축 링크 생성
- **스크린샷** — 라벨과 워터마크가 들어간 그래프 이미지로 저장
- **워터마크** — 위치와 투명도를 자유롭게 조절할 수 있는 텍스트·이미지 워터마크

### 사용자 인터페이스

- **반응형 레이아웃** — 작은 화면에 억지로 끼워 맞춘 데스크톱 뷰가 아니라, 데스크톱과 모바일 각각을 위한 전용 UI
- **다크 모드** — 라이트, 다크, 시스템 설정 따르기 중 선택
- **패널 내비게이션** — Device, Graph, Equalizer, Misc 네 패널로 기능을 정리
- **레이아웃 조정** — 데스크톱에서 구분선을 드래그해 그래프와 패널 영역의 크기를 조절
- **다국어 지원** — 영어와 한국어, 컴파일 타임 i18n(Paraglide JS) 기반
- **상단 바 커스터마이징** — 제목(텍스트·이미지·HTML), 내비게이션 링크, 브랜딩까지 자유 설정
- **[Frequency Tutorial](./features/frequency-tutorial)** — 주파수 대역의 이름과 설명을 알려주는 인터랙티브 오버레이로, 입문자에게 유용합니다.

### squig.link 연동

`*.squig.link` 도메인에서 운영하면 [squig.link](https://squig.link) 네트워크를 위한 추가 기능이 자동으로 활성화됩니다. 자세한 내용은 [squig.link 연동](./features/squiglink-integration)을 참고하세요.

- **크로스 사이트 검색** — squig.link 네트워크의 모든 사이트에서 기기 검색
- **쇼핑 링크** — 상점 데이터베이스에서 일치하는 기기를 찾으면 구매 링크를 표시
- **분석** — 다중 측정 ID를 지원하는 Google Analytics 연동
- **스폰서 콘텐츠** — 첫 방문 시 스폰서 배너 노출
